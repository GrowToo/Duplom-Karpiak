## Опис проєкту
Цей репозиторій містить програмний комплекс (Proof of Concept), розроблений у рамках бакалаврської дипломної роботи. Метою стенда є практична демонстрація криптографічних процесів та моделювання актуальних векторів атак на схеми електронного цифрового підпису (ЕЦП) та протоколи автентифікації.

Проєкт не використовує сторонніх криптографічних бібліотек (лише вбудовані модулі Python), що дозволяє «під капотом» дослідити математику процесів та їх вразливості.

## Структура комплексу
Стенд складається з 4-х основних модулів:

1. **`main.py`** — Головна точка входу. Інтерактивне консольне меню для керування модулями.
2. **`rsa_core.py`** — Базова математика RSA. Демонстрація генерації простих чисел, обчислення експонент, гешування (SHA-256) та накладання/верифікації підпису.
3. **`timing_attack_sim.py`** — Атака побічними каналами (Timing Attack). Симуляція вразливості класичного алгоритму швидкого піднесення до степеня (`Square-and-Multiply`), де криптоаналітик відновлює закритий ключ $d$ на основі мікрозатримок процесора.
4. **`emv_mitm_attack.py`** — Логічна MitM-атака на протокол EMV. Симуляція обходу вимоги FaceID (Downgrade Attack) у смартфонах шляхом маніпуляції байтами налаштувань термінала (TTQ/CTQ) у режимі Express Transit Mode.
